# Pair Eyewear SQL Project

### Project Description  
This project was completed as part of the **"Data Bases"** B.Sc course in the **Industrial Engineering & Management Department** at **Ben-Gurion University**.  

The project focuses on the business analysis of data from [**Pair Eyewear**](https://paireyewear.com), a website that sells customizable glasses and sunglasses. The main business process represented on the site is the purchase of various types of eyewear and related accessories.  

As part of the project, we designed a customized database, performed advanced SQL queries to extract meaningful insights, and created analytical dashboards that support data-driven business decisions.  

**Overall Project Grade:** 95/100
